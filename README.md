# Flutter Practice Apps 🚀

This repository contains multiple mobile application projects developed using **Flutter** and **Dart**.
The purpose of this folder is to organize and maintain different practice apps and university coursework related to mobile app development.

## 📚 Purpose

This repository is used for:

* Practicing **Flutter mobile app development**
* Learning and improving **Dart programming**
* Building **multiple mobile applications** for experimentation and practice
* Completing **university coursework related to mobile app development**
* Developing small projects to strengthen **UI design, logic building, and app architecture**

Over time, new projects will be added to this repository as part of continuous learning and development.

## 🛠 Technologies Used

* **Flutter** – Cross-platform mobile app framework
* **Dart** – Programming language used for Flutter development
* **Android Studio / VS Code** – Development environments

## 📂 Repository Structure

Each folder inside this repository represents a separate Flutter project.

Example structure:

```
Flutter-Practice-Apps/
│
├── dice_guess_number_app/
├── expense_manage_app/
├── information_app/
├── quiz_number_app/
├── student_info_panel_app/
└── more projects coming soon...
```

Every project contains its own Flutter application with its own source code, dependencies, and configuration.
